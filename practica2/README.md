## Descripcion
Modulo de gestion de productos con CRUD simulado en memoria.
Permite agregar y eliminar productos con validaciones en el formulario.

## Pantallas
- Home: pantalla de bienvenida
- Lista de productos: muestra los productos guardados
- Formulario: permite agregar un nuevo producto

## Como ejecutar
1. Abrir el proyecto en FlutLab (flutlab.io)
2. Importar el ZIP del proyecto
3. Presionar el boton Play para compilar
4. Usar la app en la vista previa
